# radio-bremsstrahlung
This repository will be used to fit free-free bremsstrahlung emission model to radio fluxes at different frequencies. If you use the programme, please cite the following paper: Hajduk, M., van Hoof, P. A. M., Åšniadkowska, K., et al., Radio observations of planetary nebulae: no evidence for strong radial density gradients, Monthly Notices of the Royal Astronomical Society, Volume 479, Issue 4, p.5657-5677, DOI: 10.1093/mnras/sty1673
ckvul.py fits the radio continuum to the data in ckvul.dat. ckvulim.py plots the fit over the data.
